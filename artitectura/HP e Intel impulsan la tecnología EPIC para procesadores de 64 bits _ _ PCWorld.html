HP e Intel impulsan la tecnología EPIC para procesadores de 64 bits

Tags: Histórico
Hewlett-Packard e Intel son los impulsores de EPIC (Explicity Parallel Instruction Computing), la tecnología en la que está basada la próxima generación de procesadores y que servirá como base para la arquitectura ISA de 64 bits.
  • inCompartir0
Manuel Cerdá

Hewlett-Packard e Intel son los impulsores de EPIC (Explicity Parallel Instruction Computing), la tecnología en la que está basada la próxima generación de procesadores y que servirá como base para la arquitectura ISA de 64 bits. El primer fruto de este acuerdo entre ambas compañías es Merced, primer chip basado en esta tecnología que comenzará a comercializarse a partir de 1999.

Según explican fuentes de ambas compañías, esta tecnología elimina las limitaciones de rendimiento de las arquitecturas RISC (Reduced Instruction Set Computing) y CISC (Complex Instruction Set Computing). Una de las características fundamentales relacionadas con este aspecto es la posibilidad de romper la secuencia natural de las tecnologías de procesadores convencionales al tiempo que las aplicaciones de software informan directamente al procesador sobre cuándo pueden realizarse las operaciones en paralelo. Otro de los rasgos importantes de EPIC es la reducción del número de conexiones no realizadas, así como los efectos de latencia entre la memoria y el procesador.

Pero quizá la más importante característica de EPIC sea lo que se ha denominado paralelismo explícito. Este término explica la posibilidad que tiene el compilador de interactuar directamente con el procesador, al contrario que las arquitecturas existentes en las cuales es necesario que el compilador cree un código de máquina secuencial antes de que se produzca la interaccción con el procesador. A este último proceso se le conoce como paralelismo implícito.
